# Keyboard-Based Attendance System using Raspberry Pi

## Description
This project is a simple attendance management system developed using Python on Raspberry Pi. 
It takes student input from a keyboard and displays output on the screen.

## Hardware Used
- Raspberry Pi 4
- HDMI Display
- USB Keyboard

## Software Used
- Raspberry Pi OS
- Python 3

## How It Works
1. User enters student details using keyboard
2. Python program processes input
3. Attendance is stored in CSV file
4. Output is displayed on screen

## Advantages
- Simple and low cost
- Easy to use
- No external sensors required

## Future Enhancements
- GUI interface
- Database storage
- RFID or biometric integration
